# Ticket-Header Layout 2

Der Ticket-Header wurde strukturell an das Mockup angepasst:

- Linker Bereich mit Icon, großem Titel `Ticket <Ticketnummer>`, Untertitel und Status-Badge.
- Rechter Bereich mit senkrechter Trennlinie, Priorität, Erstellt am und Letzte Änderung.
- `createdon` und `modifiedon` werden in der Incident-Abfrage mitgeladen.
- Cache-Buster in `taskpane.html` wurde erhöht.
